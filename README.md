# Product Dashboard – Vanilla JS

## Project Description
This project is a dynamic, client-side **Product Management Dashboard** built to streamline inventory visualization and data handling. It allows users to interact with a product catalog through a clean, responsive interface. The dashboard focuses on the core principles of **CRUD (Create, Read, Update, Delete)** operations, enabling real-time updates to the UI based on underlying data structures.

## Technologies Used
* **HTML5:** Semantic structure for accessible web content.
* **CSS3:** Custom styling featuring a modern, professional layout.
* **JavaScript (ES6+):** Vanilla JS for DOM manipulation, event handling, and logic.
* **Git/GitHub:** Version control and professional repository management.
